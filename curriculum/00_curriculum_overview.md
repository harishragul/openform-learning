# OpenFOAM Curriculum Overview

## Learning Path

```
[Module 01] Foundations
      ↓
[Module 02] First Simulations
      ↓
[Module 03] Mesh Generation
      ↓
[Module 04] Boundary Conditions
      ↓
[Module 05] Solvers & Numerics
      ↓
[Module 06] Turbulence Modeling
      ↓
[Module 07] Post-Processing
      ↓
[Module 08] Advanced Topics
```

## Estimated Time per Module

| Module | Estimated Time | Key Skills Gained |
|--------|---------------|-------------------|
| 01 Foundations | 3–5 hours | CFD basics, case structure, navigation |
| 02 First Simulations | 4–6 hours | Running cases, reading output, ParaView basics |
| 03 Mesh Generation | 6–10 hours | blockMesh, snappyHexMesh, mesh quality |
| 04 Boundary Conditions | 4–6 hours | All BC types, dictionary syntax |
| 05 Solvers & Numerics | 6–8 hours | Solver selection, convergence, schemes |
| 06 Turbulence | 5–8 hours | RANS/LES setup, wall treatment |
| 07 Post-Processing | 3–5 hours | ParaView, functionObjects, sampling |
| 08 Advanced | 10–15 hours | Custom code, HPC, optimization |

**Total**: ~40–60 hours of focused learning

## Tools You'll Need

- OpenFOAM (v2312 ESI or Foundation v11)
- ParaView (usually bundled with OpenFOAM)
- A terminal (bash/zsh)
- A text editor (VS Code recommended)
- Optional: Python 3 + PyFoam for Module 08
